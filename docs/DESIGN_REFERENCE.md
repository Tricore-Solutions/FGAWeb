# Design Reference Document (DRD)
_Based on Bayer 04 Leverkusen Website_  
_Reference URL: https://www.bayer04.de/en-us_

---

## 1. Overview

The client has selected the **Bayer 04 Leverkusen** professional football club website as the design inspiration for the FGA website. This document analyzes and documents all key design elements, layouts, and features to guide the development process.

**Design Philosophy:** Modern, bold, sports-focused with strong visual hierarchy and dynamic content presentation.

---

## 2. Color Scheme & Branding

### Bayer 04 Colors (Reference)
- **Primary:** Red (#E20613) - Bold, energetic
- **Secondary:** Black (#000000) - Strong, professional
- **Accent:** White/Light Gray - Clean contrast
- **Background:** Dark grays and blacks for sections

### FGA Color Adaptation
- **Primary:** To be defined (based on FGA brand colors)
- **Secondary:** Dark backgrounds for modern, professional look
- **Accent:** Clean whites and light colors for text and CTAs
- **Use bold, high-contrast design**

---

## 3. Typography

### Observed Typography Hierarchy
- **Hero Headlines:** Very large, bold, uppercase
  - Heavy weight (700-900)
  - High contrast color (white on dark, or accent color)
- **Section Headers:** Large, bold with accent colors
  - Example: "NEWS", "WERKSELF.TV", "BUILDING OUR FUTURE"
- **Body Text:** Clean, readable sans-serif
  - Medium weight for body copy
  - Good line height for readability
- **Labels/Tags:** Small, uppercase, colored labels for categorization

### Recommended Fonts for FGA
- **Headers:** Inter, Poppins, or Montserrat (Bold/Black weights)
- **Body:** Inter, Open Sans, or similar modern sans-serif
- **Accents:** Same as headers but in different weights

## 4. Layout Structure

### 4.1 Header/Navigation

**Key Features:**
- **Dark background** (black or very dark gray)
- **Logo on left** with team/brand name
- **Horizontal mega menu** with main navigation items:
  - Bayer 04 (About)
  - Teams
  - BayArena (Venue)
  - Project Future
  - Performance Centre
  - Membership
  - Fans
  - Sustainability
  - Business
- **Right side utilities:**
  - Language selector (EN dropdown)
  - Search icon
  - Shopping cart
  - Login/User account
  - Mobile hamburger menu
- **Prominent CTAs:** "TICKETS" and "SHOP" in accent color (red)
- **Sticky/Fixed navigation** on scroll

**FGA Adaptation:**
```
Logo + Academy Name | About | Programs | Events | Tournaments | Membership | News | Contact || Login | Register | Search | Menu
```

---

### 4.2 Hero Section

**Design Elements:**
- **Full-width image slider/carousel**
- **Multiple slides** with match photos, team celebrations, action shots
- **Overlay text on left or right side:**
  - Small category label (e.g., "BUNDESLIGA")
  - Large headline in bold (match result or news)
  - Brief description text
  - "SHOW MORE" CTA button
- **Slide indicators** at bottom (dots)
- **Auto-play carousel** with manual navigation arrows
- **Dark gradient overlay** on images for text readability

**FGA Hero Adaptation:**
- Use academy action photos (training, matches, celebrations)
- Highlight: Upcoming events, recent achievements, featured programs
- CTAs: "Register Now", "View Events", "Learn More"

---

### 4.3 News Section

**Layout:**
- **Section title:** Large, bold with accent symbol (+ icon in red)
- **Grid layout:** 4 columns on desktop
- **Card design:**
  - Large thumbnail image
  - Small category tag (colored, e.g., "Bundesliga")
  - Date stamp
  - Headline (bold, white text)
  - Brief excerpt
  - "Show more" link
- **"Show more" button** at bottom to load more articles
- **Dark background** (black) with white text

**FGA News Adaptation:**
- Categories: "Programs", "Events", "Tournaments", "Achievements"
- Show latest news, announcements, match results
- Link to full article pages

### 4.4 Video Section (Werkself.TV)

- **Section title:** "WERKSELF.TV" with large bold text and accent symbol

- **Video cards:**
  - Thumbnail with play button overlay

### 4.5 Feature Section (Building Our Future)

**Design:**
- **Full-width section** with strong visual impact
- **Large section title** with accent symbol (+)
- **Content layout:**

- **High contrast:** White text on dark background

**FGA Feature Adaptation:**

### 4.7 Statistics Section

**FGA Adaptation:**

## 5. Component Patterns

### 5.1 Cards

**Style:**
- **Dark background** (black or very dark gray)
- **No visible border** (or subtle border)
- **Hover effect:** Slight scale or brightness change
- **Content:**
  - Image at top (full width)
  - Small colored tag/label
  - Title in bold white text
  - Date/metadata
  - Brief description
  - CTA link or button

### 5.2 Buttons

**Primary CTA:**
- **Red background** with white text
- **Bold, uppercase text**
- **Rounded corners** (slight)
- **Hover:** Darker shade or slight animation

**Secondary CTA:**
- **Transparent with red border**
- **Red text**
- **Hover:** Filled red background

**Text Links:**
- **"Show more" style** with arrow icon
- **Red color** on hover
- **Underline** or color change

### 5.3 Forms

**Style:**
- **Dark background inputs** with light text
- **Clearly labeled fields**
- **Validation indicators** (red for errors)
- **Large, accessible input fields**
- **Prominent submit button**

### 5.4 Icons

**Usage:**
- **Custom SVG icons** for navigation
- **Social media icons** (brand colors)
- **Accent symbols:** + icon in red as section decorators
- **Consistent style** throughout

## 7. Responsive Design

### Breakpoints Observed
- **Desktop:** Full multi-column layouts (4-5 columns)
- **Tablet:** 2-3 columns
- **Mobile:** Single column, stacked layout

### Mobile Adaptations
- **Hamburger menu** for navigation
- **Simplified hero** with smaller text
- **Single column grids** for content
- **Touch-friendly** buttons and interactions
- **Collapsible sections** where appropriate

---

## 8. Performance & UX Considerations

### Loading & Optimization
- **Lazy loading** for images below fold
- **Progressive image loading** (blur-up effect)
- **Fast page loads** despite heavy media
- **Skeleton screens** for loading states

### Accessibility
- **High contrast** text and backgrounds
- **Clear focus states** for keyboard navigation
- **Alt text** for images
- **Semantic HTML** structure
- **ARIA labels** where appropriate

---

## 9. Animations & Interactions

### Observed Animations
- **Fade-in on scroll** for sections
- **Carousel transitions** (slide or fade)
- **Hover animations** on cards and buttons
- **Smooth scrolling** between sections
- **Loading animations** for dynamic content

### Recommended Libraries
- **Framer Motion** or **React Spring** for React animations
- **AOS (Animate On Scroll)** for scroll-triggered animations
- **Swiper.js** for carousels and sliders

## 10. Content Strategy

### Content Types
1. **News/Articles:** Match reports, announcements, updates
2. **Videos:** Highlights, interviews, behind-the-scenes
3. **Events:** Upcoming matches, special events
4. **Media:** Photo galleries, press releases
5. **Static:** About, history, facilities, contact

### Content Presentation
- **Visual-first:** Large images and videos
- **Scannable:** Clear headlines, short paragraphs
- **Categorized:** Tags and categories for organization
- **Dated:** Timestamps for relevance
- **Linked:** Cross-linking between related content

## 11. Technical Implementation Notes

### Frontend Stack (Already Defined)
- React + Vite
- TailwindCSS (perfect for this design style)
- React Router for navigation
- Axios for API calls

### Key TailwindCSS Classes to Use
```css
/* Dark backgrounds */
bg-black, bg-gray-900

/* Bold typography */
font-bold, font-black, text-4xl, text-6xl, uppercase

/* Grid layouts */
grid, grid-cols-1, md:grid-cols-2, lg:grid-cols-4

/* Cards */
rounded-lg, overflow-hidden, transition, hover:scale-105

/* Red accent (customize to FGA colors) */
text-red-600, bg-red-600, border-red-600
```

### Component Libraries to Consider
- **Swiper:** Carousel/slider
- **React Icons:** Icon library
- **Headless UI:** Accessible components (modals, dropdowns)
- **React Hot Toast:** Notifications
- **React Hook Form:** Form handling

## 16. Key Takeaways from Bayer 04 Reference

### Visual Style
✓ **Bold and dramatic** - Large text, high contrast  
✓ **Dark-themed** - Black backgrounds with white/red accents  
✓ **Image-heavy** - Visual storytelling is priority  
✓ **Clean and modern** - No clutter, good whitespace  
✓ **Professional** - Polished, premium feel  

### User Experience
✓ **Easy navigation** - Clear menu structure  
✓ **Quick access** - Important CTAs always visible (Tickets, Shop)  
✓ **Content discovery** - Well-organized sections  
✓ **Mobile-friendly** - Responsive on all devices  
✓ **Fast loading** - Optimized performance  

### Content Strategy
✓ **News-driven** - Latest updates prominently featured  
✓ **Event-focused** - Matches and events are central  
✓ **Community engagement** - Social integration  
✓ **Multimedia** - Mix of text, images, video  
✓ **Regular updates** - Fresh content keeps users coming back
